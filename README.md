2進数・16進数の変換や演算を行えるアプリ
====
　
## 開発環境  
- MacOS Catalina version 10.15  
- npm version 6.13.4 
- node version 12.16.0 

## できること
- 2進数 ⇄ 10進数 ⇄ 16進数の相互変換
- 2進数・16進数の論理演算、加算
